Yes there is, if you are part of a project, you can make an initial offering or what is called an Initial Project Offering (IPO), here you play the role of project owner, and will offer an initial decentralized token sale, or fundraising.

The method is easy, after you log in then
click on your profile section,

then click Create IDO Poll,

enter the token contract address,

fill in the IDO information,

fill in the Project information,

Submit, and confirm,

you have successfully created an IDO pool.

That's right, apart from contributing to IDO, creating an IDO pool, there are several basic features, for example locking tokens, searching for token addresses, or searching for IDO using token contract addresses.

Token locking is a token that is locked as a vesting program, this is usually done by the project owner or project team to guarantee and as proof that the token has been divested and will be able to be opened within the time period determined by them.

The method is very easy
click on your profile

then click Create Locker

fill in the details

• locker name

• token contract address

• lock amount

• withdrawer address

• set the date period for withdrawal

• click approve token

• confirm<\br>
and successful, the token has been vested.
